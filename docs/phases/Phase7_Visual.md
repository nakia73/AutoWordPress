# Phase 7: Visual（画像自動生成）詳細仕様書

**テーマ:** Visual Appeal
**ゴール:** AI画像生成により、記事のアイキャッチ画像を自動作成する
**前提:** Phase 5（MVP Launch）完了後、Betaフィードバックに基づき優先度決定

---

## 1. 目的

ブログ記事において、アイキャッチ画像はCTR向上・SEO効果・ブランディングに重要です。
本フェーズでAI生成により**記事内容に最適化された画像**を自動作成します。

---

## 2. 機能要件

### 2.1 画像生成フロー

```
記事生成 → タイトル・要約抽出 → 画像プロンプト生成 → AI画像生成 → WordPress投稿
```

### 2.2 画像生成API候補

| API | 特徴 | コスト |
|-----|------|--------|
| DALL-E 3 | 高品質、テキスト描画可能 | $0.04-0.12/枚 |
| Stable Diffusion | オープンソース | インフラコストのみ |

**初期採用:** DALL-E 3

### 2.3 ユーザー設定

- スタイル: フォトリアル / イラスト / ミニマル
- カラーテーマ: ブランドカラーに合わせる
- 画像サイズ: 1200x630px（OGP推奨）

---

## 3. 成功基準

- 画像生成成功率 95%以上
- ユーザー満足度 80%以上
- CTR向上 20%

---

## 4. 実装優先度

コスト影響があるため、プラン設計と合わせて検討。上位プランの差別化要素として位置づけ。
