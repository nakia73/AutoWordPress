# Phase 0: Mockup（マーケティング・集客）詳細仕様書

**テーマ:** Visualization & Traction
**ゴール:** サービス開発前にSNS拡散用のデモを作り、見込み顧客（ウェイトリスト）を獲得する。
**期間:** Week 1-2

---

## 1. 目的 (Purpose)

「実際に作る」前に「欲しい人がいるか」を確認します。特に X (Twitter) や Threads で話題になるような、**「URLを入れただけで魔法のようにブログができる」瞬間を演出すること**が最大の目的です。

## 2. 成果物

### 2.1 魔法のランディングページ (LP)

- **訴求:** 「Your AI Marketing Team. Setup in 3 minutes.」
- **機能:** URL入力フォーム（見た目）と、ウェイトリスト登録（メールアドレス収集）。

### 2.2 セットアップ演出 (The "Magic" Moment)

- 「分析中...」「キーワード抽出中...」「WordPress構築中...」といった、裏側で複雑な処理が走っていることを期待させるリッチなアニメーション演出。

### 2.3 管理画面シミュレーション

- 記事が自動で生成され、WordPress（風の画面）に投稿されていく様子のモックアップ。

---

## 3. マーケティング活動

- **SNS展開:** 上記の演出画面を録画し、X や Threads で「開発中のプロダクト」として投稿。
- **検証:** 50名以上のウェイトリスト登録を目指し、市場の需要を定量的に測定する。

## 4. 成功基準

- 実際の開発コストをかける前に、50人以上の「ぜひ使いたい」というターゲットユーザーの連絡先を獲得できていること。
