## プロジェクトについて

このリポジトリは **Argo Note** のプロダクト仕様を定義するためのドキュメントプロジェクトです。

- ドキュメントの完成度を高めることが目標

## タスク登録ガイド

### 新規タスクの登録方法

1. 「未着手」セクションに行を追加
2. 以下の情報を記入：
   - **ID:** T + 連番（例: T008）
   - **タスク:** 簡潔な説明（50文字以内）
   - **優先度:** 高 / 中 / 低
   - **依頼日:** YYYY-MM-DD
   - **備考:** 追加情報（任意）

### タスクのステータス変更

- 着手時: 「未着手」→「進行中」に移動
- 完了時: 「進行中」→「完了タスク」に移動、成果物を記載

### 優先度の基準

| 優先度 | 基準 |
|--------|------|
| 高 | MVP仕様に直接影響 / ブロッカー |
| 中 | 仕様の明確化に必要 |
| 低 | あれば良い / 将来対応可 |
